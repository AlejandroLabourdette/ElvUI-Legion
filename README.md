# ElvUI-Legion

ElvUI ported and adapted for World of Warcraft Legion 7.3.5.

## Personal motivation

This project was born out of the need for specific configurations that were not available in any public version of the addon. Rather than waiting for someone else to implement them, I adapted it myself according to my own preferences and playstyle.

It is not an official fork nor is it supported by the original ElvUI team — it is simply the result of covering personal needs that I could not find anywhere else.

## Free to use

Anyone is welcome to use, modify, or distribute this project. If you find it useful, go ahead.

## Installation

Copy the `ElvUI/` and `ElvUI_Config/` folders into:

```
World of Warcraft/_retail_/Interface/AddOns/
```

No compilation or additional steps are required. The WoW 7.3.5 client interprets Lua files directly.
